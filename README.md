# MotionView

## Keybinds

**Legend:** `Cmd` on macOS, `Ctrl` on Windows/Linux.

| Context | Keybind | Action |
|---|---|---|
| Global | `Cmd/Ctrl + 1` | Switch to Viewing mode |
| Global | `Cmd/Ctrl + 2` | Switch to Planning mode |
| Global | `Cmd/Ctrl + Shift + K` | Clear everything (field + plan) |
| Global | `F` | Fit/reset field position |
| Global | `Space` | Play/Pause playback (or toggle Auto‑follow Head when live‑connected) |
| Viewing | `Cmd/Ctrl + O` | Open JSON file |
| Viewing | `Cmd/Ctrl + K` | Clear Viewer |
| Viewing | `S` | Start/stop live streaming (if connected) |
| Viewing | `T` / `P` | Toggle Planned Overlay |
| Viewing | `C` | Connect/disconnect |
| Viewing | `←` / `→` | Step to previous/next pose |
| Planning | `Space` | Play/Pause plan playback |
| Planning | `Delete` / `Backspace` | Delete selected waypoint(s) |
| Planning | `←` / `→` / `↑` / `↓` | Nudge selected waypoint(s) |
| Planning | `Shift + ←/→/↑/↓` | Nudge selected waypoint(s) by 5× step |
| Planning | `Cmd/Ctrl + Z` | Undo |
| Planning | `Cmd/Ctrl + Shift + Z` | Redo |
| Planning | `Cmd/Ctrl + K` | Clear planned path |

Notes:
- Keybinds are disabled while typing in input fields.
